# Automated Grading Support System (AGSS)

AGSS is an innovative AI-driven solution developed to address the heavy workload of grading specialized essay-based exams (such as Software Testing - SWT301) at FPT University. By integrating advanced AI models, the system streamlines the manual grading workflow, ensuring high consistency, fairness, and academic integrity.

### Key Features
* Automated Essay Scoring: Minimizes manual grading time by up to 50% using standardized criteria.

* Academic Integrity Shield: Built-in modules to detect Plagiarism and AI-generated content (ChatGPT/LLMs).

* Detailed Grading Reports: Automatically generates transparent justifications for scores to support re-evaluation and student feedback.

* Workflow Optimization: Bridges the gap between the Examination Department, Heads of Subjects, and Lecturers.
